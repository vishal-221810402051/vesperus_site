   - Build command: `npm run build`

   - Install command: `npm install`
5. Deploy the project.

7. Add the custom domain `vesperus.systems` in the Vercel project settings.
8. Follow the exact DNS instructions Vercel provides for the domain.

## Required Namecheap DNS Steps

Do not change Namecheap DNS before Vercel provides domain-specific instructions.

Recommended approach:

1. Deploy the project on Vercel first.
2. Add `vesperus.systems` inside Vercel project domain settings.
3. Let Vercel generate the exact required DNS records.
4. Open Namecheap domain DNS settings.
5. Add or update only the DNS records Vercel provides.
6. Wait for DNS propagation.
7. Verify domain status inside Vercel.
8. Confirm `https://vesperus.systems` loads the deployed site.

Important:

Do not guess DNS records before Vercel provides them. Vercel may provide different instructions depending on whether the apex domain, `www` subdomain, or both are configured.
