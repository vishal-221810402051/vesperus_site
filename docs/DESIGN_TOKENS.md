# Design Tokens

## Colors

- `background-primary: #05070D`
- `background-secondary: #0B1020`
- `surface-primary: #111827`
- `surface-secondary: #0F172A`
- `border-subtle: #1E293B`
- `text-primary: #F8FAFC`
- `text-secondary: #CBD5E1`
- `text-muted: #94A3B8`
- `accent-cyan: #00E5FF`
- `accent-teal: #14F1C7`
- `accent-violet: #8B5CF6`
- `warning-amber: #F59E0B`
- `success-green: #22C55E`

## Typography

- Headings: Inter or system sans-serif
- Body: Inter or system sans-serif
- Technical labels: JetBrains Mono or ui-monospace fallback

## Layout

- `max-width: 1200px`
- `section padding desktop: 96px 24px`
- `section padding mobile: 64px 20px`
- `card radius: 24px`
- `button radius: 999px`

## Usage Notes

- Use `background-primary` as the dominant page background.
- Use `background-secondary` for major section depth.
- Use `surface-primary` and `surface-secondary` for cards, panels, proof blocks, and technical modules.
- Use `border-subtle` for default borders.
- Use `accent-cyan` and `accent-teal` for primary actions, focus states, and system highlights.
- Use `accent-violet` sparingly for restrained premium highlights.
- Use `warning-amber` and `success-green` only for clear status states.
